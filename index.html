Control Panel





Defaults to Plot Builder if no options are selected, or if the Control Panel is reset.

Select an option from the Control Panel, then press the Submit button or the button below the plot box. Checkboxes under Plot Builder allow you to focus on the kind of plot you want to generate. If no boxes are selected, it'll return any one of the plot templates available.
